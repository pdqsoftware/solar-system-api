This file contains instructions on how to use the API

The allowed verbs are described below:

GET

https://pdqsoftware-solar-system-api.herokuapp.com/solarsystem - returns all data for all solar systems

https://pdqsoftware-solar-system-api.herokuapp.com/solarsystem/id - returns all data for solar systems with an id of 'id'

https://pdqsoftware-solar-system-api.herokuapp.com/solarsystem/id/planet/pid - returns data for planet 'pid' in the solar systems with an id of 'id'

https://pdqsoftware-solar-system-api.herokuapp.com/solarsystem/id/weight - returns the weight of solar systems with an id of 'id'

PUT

https://pdqsoftware-solar-system-api.herokuapp.com/solarsystem/id - changes the name of solar system with an id of 'id' (body required)

https://pdqsoftware-solar-system-api.herokuapp.com/solarsystem/id/planet/pid - changes the name and/or weight of a planet (pid) in solar system with an id of 'id' (body required)

POST

https://pdqsoftware-solar-system-api.herokuapp.com/solarsystem - adds a new solar system dependent on request body (body required)

https://pdqsoftware-solar-system-api.herokuapp.com/solarsystem/id/planet - adds a new planet to solar system with an id of 'id' (body required)

DELETE

https://pdqsoftware-solar-system-api.herokuapp.com/solarsystem - deletes all data for all solar systems

https://pdqsoftware-solar-system-api.herokuapp.com/solarsystem/id - deletes all data for solar systems with an id of 'id'

https://pdqsoftware-solar-system-api.herokuapp.com/solarsystem/id/planet - deletes all planets for solar systems with an id of 'id'

https://pdqsoftware-solar-system-api.herokuapp.com/solarsystem/id/planet/pid - deletes planet 'pid' within the solar systems with an id of 'id'
